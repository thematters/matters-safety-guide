# 網站參考與取捨

## EFF Surveillance Self-Defense

- 網址：<https://ssd.eff.org/>
- 採用：具公信力的編輯式視覺、依情境組織知識、基本觀念與工具分流、多語入口
- 不採用：直接複製鎖頭符號或其品牌黑白配置

## Amnesty International Security Lab

- 網址：<https://securitylab.amnesty.org/>
- 採用：人權倡議的直率語氣、醒目的求助入口、大字排版與插畫式資訊節奏
- 不採用：將所有內容都處理成倡議 campaign landing page

## The Pudding

- 網址：<https://pudding.cool/2021/10/lenna/>
- 採用：scrollytelling 的章節節奏、圖文同步、讓視覺服務論證
- 不採用：為展示技術而延長捲動距離，或讓動畫取代正文

## Cloudflare Radar

- 網址：<https://radar.cloudflare.com/>
- 採用：狀態總覽、證據密度、數值與細節的層級、可追溯日期
- 不採用：深色 SaaS 儀表板外觀、即時監控感、與本站無關的複雜篩選器

## Consumer Reports Security Planner

- 網址：<https://securityplanner.consumerreports.org/>
- 採用：依個人情境產生行動計畫，以及結果可回顧的任務式架構
- 不採用：漸層、3D 人物與產品 onboarding 風格。這類語彙太接近通用 AI SaaS 模板

## 綜合結論

本站視覺骨架以 EFF 與 Amnesty 的公共利益出版品感為主，互動敘事借鏡 The Pudding，證據區借鏡 Radar，個人化任務只借 Security Planner 的功能。五個來源都只提供原則，不直接複製品牌資產、版面或文案。
