# Security Policy

請使用 GitHub 的 private vulnerability reporting 回報本站的安全問題，避免在公開 issue 留下可利用細節。

<https://github.com/thematters/matters-safety-guide/security/advisories/new>

本站不處理 Matters 主平台帳號的個案支援。帳號問題請透過 Matters 官方支援管道處理，急迫數位安全事件可聯絡 [Access Now Digital Security Helpline](https://www.accessnow.org/help/)。
